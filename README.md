Questions

1. Name of your Project
2. A little Description
3. Technology that you use
4. 3 Features about your project.

Ans:

1. This Project Name is "DigiTools-Platform"



2. It is a Premium Digital Tools e-commerce website where users can browse and
   buy products by clicking "Buy Now" button.Users can add products to the cart
   and also remove them from the cart.



3. In this project, I used the following technologies: i. React.js ii.
   Javascript (Es6) iii.Tailwind CSS iv. DaisyUI



4. Features: 

(1) Products and Cart button toggling


(2) "Buy Now" buttons Features

i.Add products to the cart

ii.Change button design after adding

iii.Update cart item count

iv.Calculate total price dynamically

v.Show toast notifications




(3) "Remove" buttons Features.

i.Remove products from the cart

ii.Update "Buy Now" button design

iii.Update cart item count

iv.Update total price after removal

v.Show toast notifications



(4) "Proceed to CheckOut" buttons Features

i.Remove all products from the cart

ii.Reset cart count

iii.Show toast notification
